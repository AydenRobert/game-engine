Creates a configuration for plane geometries given the input parameters. NOTE: vertex and index array are dynamically allocated and should be freed by the caller, should not be used in production.

Parameters
widthOverall width, must be non-zero.
heightOverall height, must be non-zero.
x_segment_countNumber of segments on the x-axis, must be non-zero.
y_segment_countNumber of segments on the y-axis, must be non-zero.
tile_xAmount texture should tile across x-axis, must be non-zero.
tile_yAmount texture should tile across y-axis, must be non-zero.
nameThe name of the generated geometry.
material_nameThe name of the material used.
Returns
A geometry configuration, can be fed into geometry_system_acquire_from_config().
